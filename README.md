# JR Marketing — Website

Professional marketing agency website built from scratch.
Live at: https://jr-marketing.netlify.app/

## What it does
A fully responsive business website for a marketing agency,
featuring service pages, workflow section, live stats dashboard
and a contact form.

## Tech Stack
- HTML5 / CSS3 / JavaScript
- Responsive design (mobile-first)
- Deployed on Netlify

## Features
- Animated stats dashboard (bereik, leads, conversie, ROAS)
- Smooth scroll navigation
- Contact form with confirmation
- Clean, conversion-focused copywriting in Dutch
- Fast loading — no frameworks, pure HTML/CSS/JS
